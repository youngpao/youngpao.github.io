##0-100以内的偶数并5个数字一行输出 ##

    <?php 
    
    for ($i=1,$n=0; $i <100 ; $i++) { 
    	# code...
    	if ($i%2==0) {
    		# code...
    		echo $i;
    		$n=$n+1;
    		if ($n%5==0) {
    			# code...
    			echo "<br/>";
    		}
    		
    	}
    }
    
    ?>

这里新增一个变量$n,是用来计数，当每增加一个偶数，$n就加1，所以每当$n是除以5的整数时，就增加一个换行。
